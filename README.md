# Solo
经过两年的开发历程，[qcip](https://github.com/cnlancehu/qcip) 项目已经趋于完善。

然而，考虑到 Golang 在性能优化方面的局限性，以及各云服务厂商 SDK 过于庞大的问题，

我们决定使用 Rust 重写整个项目，并将其命名为 Solo。

## 为什么选择 Solo?
Solo 寓意一对一的专属连接，确保服务器的关键端口仅供个人访问，这完美契合了我们的项目理念。

## 开发进展
目前项目重构正处于初期阶段，计划于寒假期间完成开发。

新版本将引入多项重要功能：

- 扩展云服务厂商支持
- 增加通知渠道选项
- 支持定时任务功能
- 提供网页配置界面
- 开放 API 接口，支持第三方应用集成
- 多语言
