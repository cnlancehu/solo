# Solo

一个保护 / 隐藏服务器端口的轻量级工具

## 功能
如果你
1. 有自己的服务器，且受到 [Solo 支持](#支持的云服务商)
2. 服务器上有一些重要的端口，例如 SSH、远程桌面、宝塔面板 等
3. 希望这些这些端口只能被自己访问，不对外暴露且不会被扫描

那么 Solo 就是你的不二之选

只需要简单几步，就能让这些端口只对你开放，其他人无法访问:

1. 在你的个人设备上安装 Solo
2. 按照文档配置
3. 运行 Solo (设置开机启动或定时运行)
