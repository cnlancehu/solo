# Solo

## [文档](https://solo.lance.fun/) | [下载](https://solo.lance.fun/zh/download/)

一个保护 / 隐藏服务器端口的轻量级工具

Solo 通过动态修改防火墙规则的 CIDR 区块为你的实时 IP 地址，确保只有你可以访问关键服务器端口，有效防止未授权的访问。
